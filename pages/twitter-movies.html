Twitter movie analytics

December 2016

This is an overview of a group project done for the Berkeley class on Data Storage and Retrieval. Our project aimed at mining data from Twitter to better understand the buzz for a movie prior to its release which can then be used for targeted marketing, merchandising and selecting release dates.

We built a prototype sourcing data from Twitter, online movie databases (IMDB), conducted basic sentiment analysis on the tweets to assess buzz on hollywood productions, and aggregated the basic analysis into a dashboard.
